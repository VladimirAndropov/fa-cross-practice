# Расписание


- запускаться как приложение в рабочей ОС , если работаете в Windows то запускаете как Windows- приложение , если в мак ОС - то iOS 

         В случае если возникают проблемы с запуском в родной среде, то используем обходной путь через эмулятор android или Chrome (в этом случае нам еще придется использовать прокси)

        dart run first/flutter_application_1/server.dart

- Приложение должно выводить на экран список преподавателей или групп (список)
- По ссылке на преподавателя или группу открывать расписание занятий (календарь)
- Формировать расписание по выбранному преподавателю
- Данные загружать по api с ruz.fa.ru (файл ruz.fa.ru.postman_collection.json)
